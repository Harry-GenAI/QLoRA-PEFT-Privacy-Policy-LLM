# Privacy Policy LLM Fine-Tuning with QLoRA and PEFT

Project 06 in my Generative AI learning series: a domain-focused language model fine-tuned with PEFT and QLoRA for privacy policy and company policy question answering.

This project uses `mistralai/Mistral-7B-Instruct-v0.2` as the base model and trains lightweight LoRA adapters on a custom JSONL instruction dataset. The model can then answer policy-related questions through an interactive terminal inference script.

## Features

- Fine-tunes a large language model using parameter-efficient fine-tuning.
- Uses QLoRA with 4-bit quantization to reduce GPU memory usage.
- Trains LoRA adapters instead of updating the full base model.
- Uses a custom `dataset.jsonl` instruction-response dataset.
- Provides a simple interactive inference script for testing responses.

## Dataset Format

The training data is stored in JSONL format. Each line contains one instruction and one expected response.

```json
{"instruction":"What is company policy?","response":"Company policy defines the rules and guidelines employees must follow within the organization."}
```

## How It Works

1. `train.py` loads the custom JSONL dataset.
2. The base Mistral model is loaded in 4-bit precision using BitsAndBytes.
3. PEFT applies LoRA adapters to selected attention projection layers.
4. The model trains only the adapter weights, making fine-tuning more efficient.
5. The trained adapter and tokenizer are saved to `proj_vi_outputs`.
6. `inference.py` loads the base model with the saved adapter and runs interactive Q&A.

## Installation

Create and activate a virtual environment:

```bash
python -m venv venv
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

If PyTorch is not already installed, install the version that matches your CUDA setup from the official PyTorch installation page.

## Training

Run:

```bash
python train.py
```

After training, the LoRA adapter files and tokenizer are saved in:

```text
proj_vi_outputs/
```

## Inference

Run:

```bash
python inference.py
```

Example questions:

```text
What is data privacy policy?
Purpose of privacy policy?
Why do companies have policies?
```

## Example Output
